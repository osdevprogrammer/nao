# nao - Not *ANOTHER* OS!

## Compilation

### Prerequisites

1. GCC 13 (MUST BE 13!!!!) (sudo apt install gcc-13)

### Instructions

1. Install required packages.

Debian/Ubuntu: `build-essential gcc-13 nasm grub-pc-bin grub-common xorriso dosfstools mtools qemu-system-x86 qemu-utils`

2. Clone this repository
3. Run `make run`.
4. Done!
